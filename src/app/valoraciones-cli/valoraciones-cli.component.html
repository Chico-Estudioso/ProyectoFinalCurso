Descripción del proyecto: (PD: Leer código de este html)

Este proyecto, está conformado por dos principales apartados entre los que se encuentran un HUD para los productos. Y un pequeño apartado para los clientes, el cual está incompleto. En el apartado de los productos tengo programada una tabla que viene de Json dicho jota son se encuentra en la carpeta sets con 20 productos distintos entre los cuales tiene un atributo que se llama valoración media, el cual en cuanto está el proyecto. Totalizado con los demás integrantes recibiré ese dato y se calculará una media dinámica para poder colocarla en el atributo de la tabla. Este atributo cambia el color Del. Texto de valoración media conforme número sea. He querido hacerlo todo esto como si fuera la vista administrador, entonces se puede modificar cualquier producto lo que pasa que el cambio no se verá reflejado en el Json para que yo pueda trabajar sin modificación hasta finalizar.
